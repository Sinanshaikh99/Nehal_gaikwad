## ğŸš€ Getting Started

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser, or
3. Use VS Code Live Server for hot reload:
   - Install "Live Server" extension in VS Code
   - Right-click on `index.html`
   - Select "Open with Live Server"

### Deploy to Vercel

1. Install Vercel CLI (optional):
   ```bash
   npm install -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

Or simply:
- Push to GitHub
- Import project in Vercel dashboard
- Deploy automatically

## ğŸ¯ Customization

### Update Personal Information

Edit `index.html` to update:
- Name and title
- About section text
- Education details
- Skills
- Experience descriptions
- Contact links

### Change Colors

Edit `css/style.css` or modify Tailwind classes in `index.html`:
- Purple: `purple-400`, `purple-600`
- Pink: `pink-400`, `pink-600`
- Blue: `blue-400`, `blue-600`

### Add Resume Download

Replace the placeholder button in the Contact section with your resume link:
```html
<a href="path/to/resume.pdf" download class="btn-secondary">
    <i class="fas fa-download mr-2"></i>Download Resume
</a>
```
